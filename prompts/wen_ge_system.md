# 文格系统 · 选择与半定制流程

## 文格选择问题组

用户选择文格后，AI 通过以下问题完成半定制：

---

### 问题组 A：书香门第

```
你出身世家，但这个世家现在处境如何？

A. 祖上是名门，如今虽然家道中落，但诗书传家的风气还在
B. 家族正处鼎盛，父亲在朝中为官，你是被羡慕的"官二代"
C. 家族表面风光，内里已经危机四伏，你在不安中长大
D. 家族早已败落，你是靠变卖旧物和亲友接济生活的落魄贵族
```

```
你在这个家族中，最看重的是什么？

A. 家族的荣誉——宁可自己受委屈，也不能丢了祖宗的脸面
B. 个人的志向——你不想活在家族的光环下，想自己闯出一片天
C. 家人之间的情感——不管外面的世界如何，家人在身边就好
D. 家族的传承——你是最后一代认真读诗书的人，有种孤独感
```

```
如果遇到一个你不喜欢但家族要求你亲近的人，你会怎么做？

A. 表面应付，内心抗拒，但不会公开反对
B. 直接拒绝，不给面子
C. 想办法让对方主动疏远自己
D. 忍着，找其他方式发泄（比如写诗）
```

### 问题组 B：江湖游侠

```
你为什么成为游侠？

A. 看不惯权贵欺压百姓，动手打了人，被迫离开家乡
B. 从小被师父收养，学了一身武艺，下山历练
C. 家门被奸人陷害，你死里逃生，从此浪迹天涯寻复仇机会
D. 天生不喜欢被束缚，向往自由，就成了现在这样
```

```
你对"侠"的理解是什么？

A. 路见不平拔刀相助，哪怕牺牲自己
B. 锄强扶弱，但要先保护好自己
C. 侠之大者为国为民，像郭靖那样
D. 随心所欲，不伤害无辜就行
```

```
你最重视的关系是什么样的？

A. 生死之交——哪怕只有一个人，能为彼此两肋插刀就够了
B. 师徒情——师父的恩情一辈子不忘
C. 自由的孤独——不需要太多羁绊，一个人也挺好
D. 义气江湖——大家聚在一起就是兄弟，不谈利益
```

### 问题组 C：谋士策士

```
你为什么选择做谋士？

B. 家学渊源，从小读兵法，有志向

D. 被迫入局——家族利益绑着你，没有选择
```

```
你对自己"谋士"身份的自豪感来源是什么？

A. 算无遗策——我的计划从未失败
B. 辅佐明主——跟对的人，做对的事
C. 洞察人心——我看人极准
D. 济世安民——我的谋略能救天下苍生
```

```
如果你的主公要做一件你判断必败的事，你会怎么做？

A. 力谏，哪怕触怒主公
B. 阳奉阴违，私下做安排降低损失
C. 沉默服从，但事后复盘反思
D. 离开，另寻明主
```

### 问题组 D：热血将军

```
你是怎么成为将军的？

A. 从小兵一步步打上来，全凭战功
B. 将门虎子，父亲是名将，我继承衣钵
C. 被逼无奈——边关告急，我临时从军
D. 被人赏识提拔，一步步走到今天
```

```
你最骄傲的一场战役是什么样的？

A. 以少胜多，绝境翻盘
B. 不战而屈人之兵，靠谋略不靠杀人
C. 保护了普通百姓，哪怕战略上不划算
D. 和兄弟们一起拼过命，无论输赢都刻骨铭心
```

```
如果你发现自己的主公其实是坏人，你会怎么办？

A. 质问主公，给他一个改正的机会
B. 沉默，但开始暗中做准备（退路或后路）
C. 直接离开，不带走一片云彩
D. 纠结——主公对我有恩，哪怕他是坏人我也……（说不下去）
```

### 问题组 E：世家闺秀

```
你对自己"世家女子"的身份是什么态度？

A. 自豪——出身名门是荣耀，也是责任
B. 厌倦——这个身份是枷锁，我想挣脱
C. 矛盾——有骄傲也有无奈
D. 无感——这个身份是别人给的，不是我选的
```

```
你在府中/家族中，最亲近的人是谁？

A. 母亲或祖母——唯一真正懂我的人
B. 兄弟姐妹——我们抱团取暖
C. 奶娘或贴身丫鬟——她比家人更像家人
D. 没有人——我是孤独的
```

```
如果你要反抗一件事，最可能是因为什么？

A. 我的婚事——我不想嫁给不认识的人
B. 家族利益——他们要我和不喜欢的人合作
C. 不公正的事——哪怕我是女子，也看不下去
D. 我在乎的人受委屈——谁欺负我的人我跟谁拼命
```

## 半定制生成文格卡

AI 根据用户选择，生成文格卡：

```
📜 你的文格

姓名：[AI生成，符合背景]
文格：[选定的文格类型]
身世：[AI根据选择生成]
性格标签：[3个核心标签]
人际倾向：[偏向哪种关系类型]
口头禅：[一句符合性格的话]
```

---

## 默认文格卡（用户快速开始用）

如果用户不想回答问题，可以直接用这个：

```
📜 默认文格

姓名：[用户自取]
文格：书香门第
身世：家道中落的世家子弟，靠祖传诗书在世间行走
性格标签：内敛、傲气、重情
人际倾向：慢热但一旦交心就极重情义
口头禅：「唉，说来话长……」
```
